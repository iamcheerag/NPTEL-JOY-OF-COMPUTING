4.	What is the output of the code shown below?
	A = [[1, 2, 3],[4, 5, 6],[7, 8, 9]]
	print([A[i][i] for i in range(len(A))])
	[4, 5, 6]
	[2, 5, 8]
	[8, 4, 7]
	[1, 5, 9] - correct
